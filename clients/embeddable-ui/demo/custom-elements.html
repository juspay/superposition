Custom Element Host Demo
A non-React host embedding <superposition-admin>
Open React host demo
API: host proxy Features:
Optional filter: the scoped filter goes into the element config payload as scope.context.
Theme: the colors here map to nested config.theme.colors, theme.button.primary, theme.icon, theme.search, theme.dropdown, and theme.banner values.
API: this host still uses the embeddable package’s REST client; the element only needs a reachable API base URL.
