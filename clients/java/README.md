# JAVA Client

## Prerequisites

- Java JDK
- Gradle 7.0 or later

## Building the Project

To build the project, run:
```
./gradlew build
```

## Running the Application

To run the application:
```
./gradlew run
```

## If having issues 
Try exporting these
```
export PATH=$JAVA_HOME/bin:$PATH
export JAVA_HOME=/opt/homebrew/opt/openjdk
```
