Quick Start

<!-- 1. Install the package -->
npm install @juspay/superposition-admin

<!-- 2. Add to your HTML -->
<div id="superposition-admin"></div>

<script type="module">
  import SuperpositionAdmin from '@juspay/superposition-admin';

  const admin = new SuperpositionAdmin({
    host: 'http://localhost:8080',
    theme: 'light'
  });

  await admin.mount();
</script>

Configuration Options

Option Type Default Description
host string window.location.origin Backend API URL
servicePrefix string '' URL path prefix
containerId string 'superposition-admin' Container element ID
theme string 'light' UI theme (light/dark/cupcake/dim)
pkgPath string './pkg' Path to WASM artifacts
